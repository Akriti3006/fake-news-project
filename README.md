# 📰 Fake News Detection Web App

A Machine Learning powered web application that detects whether a news article is **Real** or **Fake** using Natural Language Processing (NLP) techniques.

## 🚀 Live Demo

🔗 https://fake-news-project-9yzk.onrender.com

---

## 📌 Features

- Detects fake news articles instantly
- Clean and simple user interface
- Built using Flask framework
- Machine Learning based prediction
- NLP text vectorization using Scikit-learn
- Deployed online using Render

## ⚙️ Installation & Setup

### 1️⃣ Clone Repository

```bash
git clone https://github.com/Akriti3006/fake-news-project.git
```

### 2️⃣ Navigate to Project Folder

```bash
cd fake-news-project
```

### 3️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

### 4️⃣ Run the Application

```bash
python app.py
```

## 🧠 How It Works

1. User enters news text
2. Text is transformed using vectorizer
3. ML model predicts whether news is fake or real
4. Result is displayed on the webpage
